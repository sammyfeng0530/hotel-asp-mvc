# hotel-asp-mvc

*important:* 这是我早期作品，前端和兼容写得不好，但没有足够的时间修改。上传只是为了向HR展示我有基于MVC框架ASP. NET C#的开发经验。如果您是HR的话，请阅览我其他作品的前端代码，谢谢。
